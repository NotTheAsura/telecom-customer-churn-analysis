Overview

This project focuses on analyzing telecom customer churn data to identify customer behavior patterns and key factors contributing to churn. The analysis helps businesses improve customer retention strategies using data-driven insights.

Features
Data cleaning and preprocessing
Exploratory Data Analysis (EDA)
Customer churn visualization
Correlation analysis
Statistical insights generation
Business-focused reporting
Tech Stack
Python
Pandas
NumPy
Matplotlib
Seaborn
Jupyter Notebook
